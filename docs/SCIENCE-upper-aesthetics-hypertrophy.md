# SCIENCE — Upper Aesthetics (supplementary upper-body hypertrophy for endurance athletes)

**Status:** Citable grounding for the `upper_aesthetics` protocol (`upper-priority-hybrid.ts`). **Read the role correctly first (§1):** this is a **supplementary upper-body aesthetic/balance overlay for endurance athletes (even while racing) — NOT a standalone strength developer.** Its *mechanism* (hypertrophy is driven by volume/load; undulating periodization aids strength; upper-body work doesn't interfere with running) is **peer-reviewed**; its *exact dosing* is practitioner convention. Same discipline as `SCIENCE-5x5-linear-progression.md` — principle separated from convention, no fabricated authority.

**Why this doc:** the protocol cited "Schoenfeld, Helms, concurrent training research" **in a code header only**. This verifies the mechanism, flags the dosing as convention, and records the true role so the protocol is never again mis-assigned as a general strength developer (see the standalone-strength audit).

---

## 1. What it IS + what it's FOR (role correction)

A **supplementary overlay** for a runner who wants visible upper-body development without compromising running:
- **2 sessions/week**: Monday lower (loaded, but `LOWER_MAINTENANCE`-intent — supportive) + Wednesday **upper (the priority session)**; a 3rd adds a lower-durability day.
- **Upper body** runs undulating hypertrophy→strength: Base ~4×12-10 @ 62→70% 1RM, Speed ~4×8-6 @ 72→81%, peak ~3×5 @ 80-85%, plus isolation accessories.
- **Lower body** is real but secondary (squat/hip-thrust/RDL ~65-73% 1RM) — explicitly not chasing 1RM.

The header is unambiguous: *"for run-only and non-tri strength-priority plans … Designed for visible upper-body development **without compromising running**."* **True role: a supplementary upper-aesthetic add-on, by design.** It is **not** a complete strength program — its lower day is maintenance-intent and the whole structure subordinates legs to upper aesthetics. (This is exactly why the non-race builder defaults strength-develop to `five_by_five`, not this.)

## 2. The cited principle (peer-reviewed — SOUND)

- **Hypertrophy is driven by volume and effort across a wide load range.** The load–rep continuum [Schoenfeld 2017] and the dose-response of weekly volume to muscle growth [Schoenfeld, Ogborn & Krieger 2017] support the 8-12-rep hypertrophy emphasis.
- **Periodized (undulating) loading aids strength/hypertrophy** over non-periodized training [Rhea 2002 — DUP; Williams 2017 — periodization meta]. *(Honest note: DUP's superiority over linear specifically is debated; what's well-supported is that periodization beats non-periodized — see §3.)*
- **Effort-based (RIR) loading** is a validated way to autoregulate the prescription [Helms 2015; Helms 2016].
- **Upper-body strength work does not interfere with running adaptations** — concurrent interference is region- and modality-specific (greatest for lower-body, high-frequency, eccentric endurance overlap), so upper-body hypertrophy is "free" relative to run training [Wilson 2012; see `SCIENCE-concurrent-training-interference.md`].

## 3. What's convention (flag honestly)

- The **exact DUP curve** (4×12@62% → 4×6@81% → 3×5@80-85%) and rep/load steps — practitioner programming, **no per-figure source.**
- The **upper-priority / lower-maintenance split** and the Mon/Wed/Fri structure — program design.
- **DUP-specifically-better-than-linear** is *not* settled evidence; the safe peer-reviewed claim is "periodized > non-periodized." Cited as such; the undulating choice is convention.

## 4. Honest grading

| Claim | Grade |
|---|---|
| Hypertrophy from 8-12 reps + sufficient volume | **SOUND** (Schoenfeld 2017 ×2) |
| Periodization aids strength/hypertrophy | **SOUND** (Williams 2017); DUP-vs-linear specifically = **CONVENTION** |
| RIR-based autoregulation | **SOUND** (Helms 2015/2016) |
| Upper-body work doesn't interfere with running | **SOUND** (Wilson 2012; concurrent doc) |
| The exact DUP %/rep curve + the priority split | **CONVENTION** — calibrated by response, no source |
| **Role: standalone developer** | **FALSE by design** — it is supplementary (the audit correction) |

**Bottom line:** sound hypertrophy/periodization mechanism, convention dosing — and, most importantly, a **supplementary upper-aesthetic overlay, not a standalone strength program.** Use it for what it's for.
